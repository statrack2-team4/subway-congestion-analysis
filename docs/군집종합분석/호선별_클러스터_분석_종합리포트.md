# 🚇 2, 4, 5호선 클러스터 분석 종합 리포트

## 📊 분석 개요

### 분석 대상
- **2호선**: 52개 역 (순환선)
- **4호선**: 27개 역 (남북 축)
- **5호선**: 57개 역 (동서 축)
- **총 136개 역**

### 분석 방법
- **K-Means 군집 분석** (각 호선당 3개 클러스터)
- 시간대별 혼잡도 패턴을 기반으로 유사한 역 그룹화
- 각 클러스터별 특성 및 운영 전략 도출
